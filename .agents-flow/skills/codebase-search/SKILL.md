# 代码库搜索技能

你可以使用以下模式在代码库中搜索：

## 搜索策略

### 1. 符号搜索
当你知道具体的函数名、类名或变量名时，直接搜索该符号。

### 2. 模式搜索
当你知道代码模式但不确定具体名称时：
- 使用正则表达式搜索
- 搜索相关的导入语句
- 搜索类型签名

### 3. 文件搜索
当你知道文件名模式时，使用 glob 模式搜索。

### 4. 语义搜索
当你不确定具体位置但知道功能描述时，使用语义搜索。

## 最佳实践

- 优先从最具体的锚点开始搜索
- 使用多个搜索工具并行查询以提高效率
- 先读取大段代码建立上下文，再逐步细化
- 注意区分类型定义和实现
- 验证搜索结果时，检查导入路径和文件位置
